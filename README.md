Student Performance Dashboard

A Java Swing application that analyzes and visualizes student performance across multiple subjects using charts and tables. This dashboard provides key insights into student grades, pass/fail statistics, top performers, and study hours.

Features
CSV Data Import: Reads student scores from a CSV file.
Key Performance Indicators (KPIs):
Total students
Average score
Pass rate
Top student

Charts:
Bar chart: Average score per subject (Maths, English, Science, Social)
Pass/Fail chart: Shows the number of students passing or failing each subject
Grade distribution pie chart: Counts and percentages for grades A–F

Tables:
Top students ranked by average score
Full student CSV content for reference
Study Hours Analysis: Visualizes correlation between study hours and performance

Getting Started

Prerequisites
Java JDK 17+
IntelliJ IDEA or any Java IDE
JFreeChart library (jfreechart-1.5.6.jar)
OpenCSV library (opencsv-5.7.1.jar)
